11.3.203 set_input/1   ISO

Synopsis

set_input(+Stream)

makes Stream the current input stream.

Arguments

Stream

stream_object, must be ground

A valid input stream.

Exceptions

Stream errors (see ref-iou-sfh-est).

see/1, ref-iou-sfh.
