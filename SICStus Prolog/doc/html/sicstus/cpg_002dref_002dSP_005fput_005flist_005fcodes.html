12.3.77 SP_put_list_codes()

Synopsis

#include <sicstus/sicstus.h>

int
SP_put_list_codes(SP_term_ref term,
                  SP_term_ref tail,
                  char const *s);

Assigns to term a Prolog code list represented by the encoded string s, prepended to the value of tail.

Arguments

term

The SP_term_ref to be assigned

tail

The tail of the code list

s

The string to convert

Return Value

Zero if the conversion fails (as far as failure can be detected), and a nonzero value otherwise.

See Also

Creating Prolog Terms.
