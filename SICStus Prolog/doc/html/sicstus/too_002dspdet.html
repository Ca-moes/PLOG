13.4 spdet — Determinacy Checker

Synopsis

% spdet [-r] [-d] [-D] [-i ifile] fspec…

Description

The determinacy checker can help you spot unwanted nondeterminacy in your programs. This tool examines your program source code and points out places where nondeterminacy may arise.

Options

-r

Process files recursively, fully checking the specified files and all the files they load.

-d

Print out declarations that should be added.

-D

Print out all needed declarations.

-i ifile

An initialization file, which is loaded before processing begins.
