12.3.48 SP_is_float()

Synopsis

#include <sicstus/sicstus.h>

int
SP_is_float(SP_term_ref term);

Determines whether the value of term is a Prolog float.

Arguments

term

The SP_term_ref to be inspected

Return Value

1 if it is a float and 0 otherwise.

See Also

Testing Prolog Terms.
