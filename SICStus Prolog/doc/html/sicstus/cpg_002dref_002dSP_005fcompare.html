12.3.5 SP_compare()

Synopsis

#include <sicstus/sicstus.h>

int
SP_compare(SP_term_ref x,
           SP_term_ref y)

Compares two terms.

Arguments

x

The one term to compare

y

The other term to compare

Return Value

-1 if x @< y, 0 if x == y, and 1 if x @> y.

See Also

ref-lte-cte, Unifying and Comparing Terms.
